Create BodyFocusCore Package

Set up the core Swift package with models and logic

Open Xcode and create a new Swift Package:

  1. Open Xcode
  2. File → New → Package...
  3. Choose "Library" template
  4. Name it BodyFocusCore
  5. Save in your project directory
Tip: Set minimum platform to iOS 17 in Package.swift 
// Package.swift
let package = Package(
    name: "BodyFocusCore",
    platforms: [.iOS(.v17)],
    products: [
        .library(
            name: "BodyFocusCore",
            targets: ["BodyFocusCore"])
    ],
    targets: [
        .target(name: "BodyFocusCore"),
        .testTarget(
            name: "BodyFocusCoreTests",
            dependencies: ["BodyFocusCore"])
    ]
)

Define Core Models

Create the foundational data structures

Create the following model files in Sources/BodyFocusCore/Models/:

File Description Key Types
Enums.swift All enum types PrimaryGoal, KneeProfile, MuscleGroup, EquipmentType, KneeLoadLevel, DayType
UserProfile.swift User profile model UserProfile
Exercise.swift Exercise definition Exercise
WorkoutSession.swift Workout session data WorkoutSession, WorkoutExercise
CheckIn.swift Pre-workout check-in PreWorkoutCheckIn
5

Implement Workout Engine

Create the rules-based workout generation logic

Create the workout engine in Sources/BodyFocusCore/Engine/:

Key Functions to Implement:
  • generateWeek(for:startDate:) - Generate 3-day workout plan
  • generateSession(for:dayIndex:date:) - Create single workout session
  • isAllowed(exercise:for:) - Filter exercises by knee profile
  • adjust(session:with:) - Adjust workout based on check-in
Critical: Knee filtering is non-negotiable. Sensitive users get NO high knee-load exercises. Restricted users get ONLY low knee-load exercises.
